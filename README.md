# 🎈 Balloon Pop

- A cute 2D arcade game inspired by Flappy Bird, built in **Unity 6.5**.
- In Balloon Pop, a little pink balloon escapes from Kirby and floats into the night sky.
- Your mission is to keep the balloon alive by avoiding dangerous storm clouds. One collision... and POP! 💥


## 🎮 Gameplay

- Press **Start** to begin.
- The balloon gently floats into the sky.
- After a short animation, you gain control.
- Press **Space** or **Left Mouse Click** to make the balloon fly upward.
- Avoid the moving storm clouds.
- Every cloud you successfully pass increases your score.
- If you hit a cloud, the balloon bursts and the game ends.


## Getting Started

- Clone or download this repository.
- Open the project folder in Unity Hub using Unity 6.5 (or the closest compatible version).
- If prompted, import TMP Essential Resources via Window → TextMeshPro → Import TMP Essential Resources.
- Open SampleScene and press Play to test in the Editor
